Generate your own Minecraft Mod-Installer!

This is a simple tool to generate your own Minecraft Mod-Installer. You can use it to install mods on your minecraft installation without the need to install forge or fabric. Just download the installer and run it. It will install the mods for you.

Which forge version do you want to use? read how
You have to provide a URL to a forge-installer.jar file.

Forge download page recommended

Visit the forge download page and select the version you want to use. Then click on the Installer button at recommended. Wait until the Skip-Button appears and right-click on it. Choose "Copy link address" and paste it into the input field below.


Self-hosted

If you want to host the forge.jar file yourself you can provide a link to the forge.jar file. For security reasons we warn users about an installation from an untrusted source.

https://example.com/forge.jar

Where are the mods located? read how

You have to provide a URL to a zip file that contains all the mods you want to install. The zip file can not contain any folders. All files have to be in the root of the zip file.


mods.zip
├── mod1.jar
├── mod2.jar
└── mod3.jar

How do you want to handle errors? read how
Online error handling recommended

This option will open the error page on my website if an error occurs. This page will show you the error and how to fix it (if possible).

Local error handling

This option will show the error in the installer itself. This is not recommended because the error message is not very detailed and doesn't show you how to fix it.
